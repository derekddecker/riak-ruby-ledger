# Riak-Ruby-Ledger

A PNCounter CRDT with ledger transaction ids for write idempotence

## Installation

Add this line to your application's Gemfile:

    gem 'riak-ruby-ledger'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install riak-ruby-ledger
